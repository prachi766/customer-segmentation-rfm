# Customer Segmentation using RFM Analysis (Task 15)

## Objective
To perform customer segmentation using Recency, Frequency, and Monetary (RFM) analysis
and derive business actions for targeted marketing strategies.

## Steps Performed
- Created transaction dataset in Python.
- Calculated Recency, Frequency, and Monetary metrics.
- Generated RFM scores using quantile binning.
- Assigned customer segments such as Champions, Loyal, and At Risk.
- Exported segmentation results to CSV.
- Documented marketing actions for each segment.

## Files Included
- task15_rfm.ipynb
- rfm_segments.csv
- segment_actions.txt

## Outcome
Demonstrates practical customer segmentation and data-driven marketing decision skills
using Python and pandas.
